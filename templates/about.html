About Portal

Welcome to Portal

Portal is a modern, open-source community platform designed for creating, sharing, and discovering amazing content. Whether you're a writer, developer, designer, or entrepreneur, Portal provides the perfect space to connect with like-minded individuals and build your community.

Features
  • User Profiles & Authentication
  • Create, Edit & Delete Posts
  • Like System
  • Public Posts & Statistics
  • REST API
  • Responsive Design
  • Secure Authentication
Technology Stack

Backend: Flask (Python)

Database: SQLAlchemy with SQLite

Frontend: Bootstrap 5 + Modern CSS

Hosting: Render (Cloud Platform)
